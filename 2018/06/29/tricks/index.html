Tips

  • 1可以用teamviewer连接操作机 https://www.teamviewer.com/zhCN/
  • 在电脑和操作机种都安装这个软件（不会很卡也能复制粘贴，方便做题）
  • 做题网址是靶机的ip

  • 做完一题后关闭网页后靶机不会关闭，所以只要换下一题，修改网址

    0x1 key就是…

  • 查看源代码

0x2 base64

  • 火狐的hackbar可以解码base64

0x3 utf-7

  • 去除+ -，base64解码能得到key

0x4 JS challenge

  • 在源代码中看到script，js脚本，url编码格式，用hackbar解码，简单if判断

0x5 技术宅拯救世界

0x6 这是黑阔经常干的事

  • 添加后登陆即可

0x7你能找到账号密码吗？

  • 另存为网页中的图片，gif图，多以很可能在另一帧的图片中隐藏信息，Stegsolve.jar 一个图片隐写工具
  • file -> open,打开图片
  • Analyse -> Frame Browser,打开后点击下面的左右箭头，找到登陆密码
  • Analyse -> File Format查看图片数据，最下面有用户名，回到网页中登陆得到key

0x8老规矩 你懂的

  • 右键查看源代码

    0x9文件上传

  • F12，打开控制台，找到script标签的html代码,选中右键编辑
  • 新建一个txt，改成asp后缀，上传

0x10 CSS OR XSS

-右键查看源代码，发现有一个default.css，有前面的描述有些联系，点击查看

0x11 找找看看

  • 下载网站上的图片，用winhex打开，在最下面有url编码
  • hackbar解密得到网盘网址
  • 密码在源代码中，同样用hacbar解密

0x12 只能上传图片哦

  • 修改后缀名phP，再次上传，成功得到flag

0x13 照猫画虎

url?id=ket1key1111:/1/1key

  • 构造匹配正则的字符，用id以get方式传入

0x14 拍好队，绕过去

  • url/?uname[]=1&&passwd[]=2
  • 题意不是很明确，记个payload吧，原理是php对数组MD5加密的值为空

0x15 头都大了

  • 题目提示，F12打开后选中网络，F5刷新获取报文，在报文的头部字段

0x16 仔细找找呗

  • 网页源代码中有base64编码，解码即可

0x17 怎么回事，我是个flag

  • 查看源代码
  • url/?hello=welcome

0x18 flag是什么呢？

  • php黑魔法，对数组strcmp()为空
  • url?password[]=1

0x19 找呀找呀找不到

  • url?id=true
